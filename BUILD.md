# How to Build GreenLib

## Compiler

Java Jdk 8

## Libraries

spigot-1.8.8.jar

spigot-1.9.jar

spigot-1.10.2.jar

spigot-1.11.2.jar

spigot-1.12.2.jar

## Build Tool

Ant 1.10.3

## Build

* Create Folder lib

* Put all Libraries in the folder

* Run this Commands

    `ant make-spigot-old`
	
	`ant build`
